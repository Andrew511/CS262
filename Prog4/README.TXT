PROJECT TITLE:MyDate Tester
PURPOSE OF PROJECT:To test MyDate classes for situations they should be able to handle given their methods and report whether they pass or fail the tests.
VERSION or DATE: 11/27/16
HOW TO START THIS PROJECT: Open the Bluej package, then right click on the MyDateTester class and select Test all.
AUTHORS: Andrew Wrege
USER INSTRUCTIONS: For general use, follow how to start this project, if you need to change the MyDate class that is being tested, open the MyDateTester class and change the
 class name in the first import statement to the MyDate class of your choice that is loaded as a library.
